# Project Description 
This is our Project to detect the eye of the driver using OpenCV Python and alert the driver by alarm sound and notification in node-red by connecting to IBM Cloud.



# Softwares / Tools Used
- Python
- OpenCV
- Node-Red
- IBM Cloud
